Designer Mode

Copy the designer_mode file into the parent directory of this folder. 
This enables Designer Mode, showing all system icons, enabling screenshot with Home+R2 and allowing to manual restart of MainUI with Hone+L2.
Screenshots are saved in the Saves/screenshot/ folder.
Restarting MainUI allows you to edit theme assets in real time using Samba, restart MainUI, and see the changes immediately.
